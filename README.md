# 🍽️ Restaurant Management System

A backend system built using **.NET 8, Entity Framework Core, and PostgreSQL**.

## 🚀 Features

* User management (Admin / Staff)
* REST API with ASP.NET Core
* PostgreSQL database integration
* Entity Framework Core with migrations
* Clean project structure

## 🛠 Tech Stack

* .NET 8 (ASP.NET Core)
* Entity Framework Core
* PostgreSQL
* C#

## 📦 Setup

```bash
git clone https://github.com/chriscorteling/Restaurant-Management-System.git
cd Restaurant-Management-System/server
dotnet restore
dotnet run
```
